## 添加背景图片设置图片布局

引入案例响应式布局算法，lib-flexible.js

```
header {

  height: 1rem;

  width: 100%;

  h1 {

    font-size: 0.275rem;

    color: #fff;

    text-align: center;

    line-height: 0.88rem;

  }

}

div {

  background: url('~@/assets/bg2.jpg') top center no-repeat;

  background-size: 100%;  // 将图片完全显示

  font-size: 0.625rem;

  height: 90vh;

}

  

datav {

  height: 90vh;

}
```
